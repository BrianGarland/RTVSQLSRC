# RTVSQLSRC
Retrieve the SQL source for a database object

### Installation 

1. `git clone` this repository
2. Run `gmake` (available from yum)

### Usage instructions

1. `ADDLIBLE RTVSQLSRC` (or whatever library you used)
2. `RTVSQLSRC` and prompt for parameters.
